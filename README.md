# Scratch Converter / Scratch 转换器

#### 介绍

Scratch 转换器可以把 .sb / .sb2 / .sb3 类型的文件转换为 HTML 网页格式，可以离线直接打开运行，便于存储与分享。
目前提供了 Windows 和 Mac 的版本，已经添加了自动更新功能，未来会陆续升级更多小功能。

#### 慢慢会更新的功能

- [ ] exe 可运行文件转换
- [ ] 更好看的应用程序图标
- [ ] 完全中英文的支持
- [ ] 多文件同时转换（考虑中）
- [ ] 可自动发布到 CDN 网络，用户直接获取网页地址
